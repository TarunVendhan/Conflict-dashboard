# Conflict Risk Dashboard

A minimalist web-based dashboard for forecasting geopolitical conflict risk using region-specific triggers and market signals. Built with HTML, CSS, and JavaScript, this project is designed for clarity, speed, and academic precision. Ideal for law, economics, and machine learning enthusiasts.

## Features
- Region cards with conflict probability
- Market indicators (stocks, crypto, oil)
- Strategic triggers and recommendations
- Clean, black-and-white layout
